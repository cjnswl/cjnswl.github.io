# 安装Hexo

**在准备部署的目录首先安装hexo，必须保证目录下为空**

`npm i hexo-cli -g `安装

`hexo -v`验证安装结果

`hexo init`初始化

`npm install`安装组件

`hexo g`生成静态网页

`hexo s打开本地服务器`
